edu-mesh

Plataforma educativa híbrida · offline · P2P

Descarga una vez. Se reparte sola, sin internet.

Catálogo educativo curado y firmado, que se distribuye entre los dispositivos de un salón por WiFi local — sin consumir datos y sin instalar nada en el celular.

Curado y firmadoSolo contenido oficial y sin alterar (Ed25519).

1 · Descargar la app

Instálala en una computadora de la escuela (el “nodo central”). Necesita Node 18 o superior (cualquier versión).

Opción A — clonar con git 
git clone https://github.com/cesarMath27/edu-mesh.git
cd edu-mesh
npm install
Opción B — descargar ZIP
Descargar edu-mesh (.zip)

Luego, doble clic en iniciar-demo.bat (o npm run demo): muestra una URL + código QR para que los celulares entren.
